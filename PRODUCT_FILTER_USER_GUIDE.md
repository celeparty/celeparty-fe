## â“ Troubleshooting

### Q1: Filter tidak muncul di desktop
```
Solusi:
1. Refresh browser (Ctrl+R atau Cmd+R)
2. Clear cache browser
3. Coba browser lain
4. Jika masih tidak muncul, contact support
```

### Q2: Tombol reset tidak muncul
```
Penjelasan:
- Tombol reset HANYA muncul jika ada filter aktif
- Jika tidak ada filter aktif, tombol tidak ditampilkan
- Ini normal dan by design

Cara aktifkan reset:
- Gunakan minimal 1 filter (pilih jenis event, lokasi, dll)
- Tombol akan muncul otomatis
```

### Q3: Harga tidak ter-format ke Rupiah
```
Solusi:
1. Pastikan hanya input angka (0-9)
2. Jangan input tanda baca atau huruf
3. Contoh benar: 1000000
4. Sistem akan otomatis format ke Rp 1.000.000
```

### Q4: Kategori scrolling tidak berfungsi
```
Solusi:
1. Pastikan pointer mouse di dalam area kategori
2. Gunakan scroll wheel atau swipe di mobile
3. Tidak semua kategori muncul? Coba scroll dengan bar di kanan
4. Jika tetap tidak bisa, refresh halaman
```

### Q5: Filter tidak mengubah hasil produk
```
Kemungkinan:
- API sedang loading (tunggu beberapa detik)
- Tidak ada produk dengan filter kombinasi tersebut
- Internet connection lemah/terputus

Solusi:
1. Tunggu 3-5 detik untuk API response
2. Coba kombinasi filter lain
3. Check internet connection Anda
4. Refresh halaman browser
5. Contact support jika masih bermasalah
```

### Q6: Mobile toggle tidak berfungsi
```
Solusi:
1. Refresh halaman
2. Check browser size (harus < md breakpoint)
3. Coba clear cache browser
4. Gunakan browser lain untuk test
```
